## Projeto Smart TV 

   
   Projeto  de smart tv com comandos basicos, manipunado comandos do usuario.
   
   * usuario ligou e desligou a TV 
   * usuario muda 3 vezes de canal
   * usuario aumenta o volume 4 vezes enseguida diminui
